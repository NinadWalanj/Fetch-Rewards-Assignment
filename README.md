## Prerequisites

- **Node.js** v16+ and **npm** installed

## Getting Started

1. Clone the repo
   
   git clone https://github.com/NinadWalanj/Fetch-Rewards-Assignment.git

   cd fetch-dogs-app

3. Install dependencies
   
   npm install
   or
   yarn
   
4. Run the dev server

   npm run dev
   or
   yarn dev
